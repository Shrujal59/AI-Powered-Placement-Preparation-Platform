# AI-Powered-Placement-Preparation-Platform
A full-stack web application that helps students prepare for placements through profile management, resume analysis, aptitude practice, company preparation, and mock interviews.
## Tech Stack
### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Django
- Django REST Framework
- JWT Authentication

### Database
- PostgreSQL

## Completed Features
- User Registration API
- JWT Login Authentication
- Student Profile Management
- PostgreSQL Integration
- React Frontend Setup

## Project Status
Currently under development.
